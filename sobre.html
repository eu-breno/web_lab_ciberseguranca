Sobre esse trabalho

Este projeto foi desenvolvido como parte da atividade de curricularização da disciplina de Web Lab (CCC277), seguindo a proposta de transformar o conteúdo das cartilhas de segurança da informação em um website interativo.

O objetivo foi criar um site que, além de transmitir informações importantes sobre segurança digital, também apresenta um layout responsivo, páginas organizadas, além de ter a implementação de recursos como o modo Dark/Light Theme e a opção de acessibilidade, que permite aumentar o tamanho das fontes.

O site conta com pelo menos três páginas principais: a página inicial com o conteúdo sobre cibersegurança, a página "Sobre" (página atual), com a apresentação do conteúdo abordado, e uma página de contato com formulário funcional que exibe um alerta ao ser enviado. Todo o código foi estruturado com HTML com tags semânticas e CSS e JavaScript separados em pastas próprias, além de imagens organizadas e o uso de uma fonte customizada do Google Fonts.

Confira o arquivo do site no GitHub.
